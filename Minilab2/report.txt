For this minilab. We implemented a real-time image processing pipeline on the DE1-SoC that transforms raw Bayer data from the D5M camera into a grayscale edge-detected display. The design uses a series of line buffers to cache incoming pixel rows, allowing the module to access the 3-by-3 local neighborhood required for spatial filtering. We first converted the Bayer pattern to grayscale, then applied a 2D convolution engine using the kernels. To handle the signed results of the gradients, we integrated an absolute value stage before outputting the 8-bit grayscale values to the SDRAM and VGA controller. Verification was performed using a ModelSim testbench that simulated row-buffering latency and confirmed the mathematical correctness of the convolution against known pixel patterns. While edge-case pixels were handled by simple truncation, the resulting hardware successfully demonstrates high-throughput filtering, effectively utilizing the FPGA’s ALMs for logic and Block RAM for line storage.
